## About

I am a Full Stack Developer and Generative AI Engineer focused on building secure, production-minded web applications and AI-powered platforms. My work sits at the intersection of software engineering, applied AI, and product execution: I design full-stack systems, connect clean APIs to reliable data layers, and use LLMs, RAG pipelines, and machine learning models to make workflows faster and more intelligent.

I work primarily across the modern JavaScript and TypeScript ecosystem with React, Next.js, Node.js, Express, PostgreSQL, Prisma, Redis, Docker, and CI/CD tooling. On the AI side, I build with Python, FastAPI, LangChain, ChromaDB, LLM APIs, prompt engineering, data preprocessing, feature engineering, and model evaluation. My projects emphasize authentication, RBAC, scalable backend architecture, automation, measurable impact, and practical user value.

**Open To**

Full-stack engineering internships, AI/ML engineering roles, generative AI product work, open-source collaboration, hackathons, and research-driven engineering projects.
